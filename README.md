# Animal Matching Game

A simple matching game built in C# as part of practising fundamental concepts like event handling, UI interaction, and basic game logic.

This project is based on an exercise from *Head First C#*, adapted and completed for learning purposes.

---

## Features

- Click-to-reveal card mechanic  
- Match detection  
- Basic game state handling  
- Simple UI built using Visual Studio and C#
